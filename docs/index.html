Robust & Secure

Engineered for Power Users

Say goodbye to complex server routes and unsecure password databases. EmailAgg handles the aggregation for you securely.

Direct API Sending

Sends forwarded emails directly from your own Gmail or Microsoft account via official REST APIs. No third-party SMTP relays or spam risks.

Smart OTP Parser

Extracts numeric and alphanumeric verification codes automatically. Displays the OTP as a copyable badge in Telegram so you don't even have to open the message.

Granular Rules Engine

Create routing rules based on subject keywords, domains, exact senders, or body content. Scope rules to specific accounts or globally.

Fernet AES Encryption

We do not store passwords. Google and Microsoft accounts use OAuth 2.0. Custom IMAP account passwords are encrypted at rest with Fernet cryptography.

Redis Rate Limits

Includes an automatic hourly notification and forwarding limit (up to 50/hour) to prevent email forwarding loops and secure IP reputation.

Fast Sync & History API

Leverages Gmail History API for incremental updates. Instead of scanning the entire inbox, only fetches changes since the last sync history ID.

SaaS Stack

System Architecture

A decoupled microservice framework built to deploy smoothly on any cloud provider via Docker Compose.

01

FastAPI Backend

Exposes robust, rate-limited REST endpoints for the dashboard and bot. Authenticates users with JWT bearer tokens.

02

Celery Workers

Concurrency pool powered by Redis. Dispatches async synchronization tasks, notification alerts, and API forwarding triggers.

03

Next.js Dashboard

Built with TailwindCSS & TypeScript. Clean state management for managing connected mailboxes, rules, and alert preferences.

04

aiogram Bot

Asynchronous Telegram bot. Provides `/start`, `/connect`, `/rules`, and `/settings` commands with secure deep linking.
