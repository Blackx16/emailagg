```bash

```

### Step 2: Check current system status

```bash
docker compose ps
docker compose logs --tail=50 backend
docker compose logs --tail=50 worker_sync
```

### Step 3: Identify the issue

Review logs for errors. Common patterns:
- `ConnectionRefused` → Redis or PostgreSQL is down
- `Task expired` → Celery worker overload
- `IMAPError` → External IMAP server issue

### Step 4: Apply the fix
